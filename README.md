# qedmaen-business-qard
### An interactive business card for a certain Senior QA Engineer.
Welcome to BusinessQard project repository. You're in the right place if:
* You want to view my CV as a web-page
* You are interested in my UI test automation skills and QA workflow

This is a constant work-in-progress project: as I learn more and more practices and standards of UI coverage, I'm doing my best to encorporate them here.
I also have a separate project for API automation, which is currently in design phase.

## Repository structure
The repository is broken down into the following folders:
* **.github** for setting up the project processes and operations
* **src** is where the CV page source files are located
* **tests** is the folder with UI automation tests

## Project scope
In this project I am planning to achieve the following milestones:
1. Wireframe
2. Static web page
3. Interactivity
4. Public hosting
5. UI test scripts
6. CI/CD for test automation
